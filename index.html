Mental Health America of Georgia Website Redesign

MHA is a mental health non-profit with the goal of providing education, resources, and assistance to Georgian's.

The team redesigned the existing website's landing page for desktop and mobile and re-structured and organized the navigation as well.

Within the team, I was the Project Manager ensuring everything was done in a timely manner. I organized and defined tasks each person needed to conduct and came up with a revamped information architecture for easier navigation. I also conducted usability tests, user interviews, and user research.

Heuristic Evaluation

During user research, we conducted heuristic evaluations of the existing website through which we gathered insight into how we could update the site.

Problem Statement

It was observed that the user interaction of the website could hinder people from accessing relevant information, seeking the help they need, and getting connected with resources. The website did not have a search bar as well.

Solution

A simpler and consistent design across devices as well as simplified navigation to ensure people can get the mental health assistance they need.

Site Map

From user research, user interviews, and heuristic evaluations, a site map was constructed to define the flow of navigation throughout the website.

Style Tile

We wanted the website to be simple and easy to use. Existing colors were kept as they brought a sense of cheer and calm and are part of the state color's of Georgia. Button states were kept simple so as not to distract from page content.
